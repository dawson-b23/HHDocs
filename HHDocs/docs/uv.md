# General 

## Installation 
macOS and Linux
```bash
  curl -LsSf https://astral.sh/uv/install.sh | sh
```

Windows
```bash
  powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

## Commands

* `uv init [project name]` - Create a new project. If you include a name, it will create a new dir. 
Otherwise, it will use the current working dir.
* `uv add [package name]` - Start the live-reloading docs server.
* `uv run [name]` - Build the documentation site.
* `uv sync` - Print help message and exit.
